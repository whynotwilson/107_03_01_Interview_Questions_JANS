2.描述 "reset" 在 CSS 中做什麼、如何用？
	
因各家瀏覽器CSS設定不太相同，導致可能在chrome看正常，
用IE看可能就跑版，因此，世界知名的 CSS 大師 
「Eric A. Meyer」整理出一個很棒的解決方法「Reset CSS」，
針對CSS語法最容易出問題的部分，例如margin外間距、
文字大小、行高等等做統一設定，只要掛上一段
「Reset CSS」語法，就可以讓各大瀏覽器統一，防止跑版。
詳解 reset

4.CSS 引入的方式有哪些？link 和@import 的區別？
	
1.打在head底下
2.link
3.@import

差別1：link 屬於XHTML標籤，而@import完全是CSS提供的一種方式。
       link 標籤除了可以加載 CSS 外，還可以做很多其它的事情，比如定義 RSS
       ，定義 rel 連接屬性等，@import只能加載CSS。

差別2：加載順序的差別。當一個頁面被加載的時候(
       就是被瀏覽者瀏覽的時候)， 
       link 引用的 CSS 會同時被加載，而@import引用的 CSS 
       會等到頁面全部被下載完再加載。所以有時候
       瀏覽@import加載 CSS 的頁面時一開始會閃爍一下，網速慢的時候挺明顯。

差別3：兼容性的差別。由於@import是CSS2.1提出的，所以較老的瀏覽器不支持，
       @import只有在 IE5 以上的才能識別，而 link 標籤無此問題。

差別4：使用 dom 控制樣式時的差別。當使用 javaScript 
       控制 dom 去改變樣式的時候，只能使用 link 標籤，
       因為@import不是 dom 可以控制的。

詳解 link & @import
